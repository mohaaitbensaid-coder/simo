# 🚲 Taroudant Bike - Rental Service 🌴

**Taroudant Bike** is a modern and responsive landing page designed for a bike rental service in the beautiful city of Taroudant, Morocco. It provides a seamless experience for visitors to explore available bikes, view rental pricing, and book their rides directly via WhatsApp.

---

## ✨ Features

- **🚲 Professional Bike Showcase**: Display of available bike models with high-quality images.
- **💰 Dynamic Pricing Table**: Clear and transparent rental offers for various durations (from 1 hour to a full day).
- **📅 Interactive Booking Form**: Simple form to collect user details, bike preference, and rental duration.
- **💬 WhatsApp Integration**: Automated booking process that sends reservation details directly to the owner's WhatsApp.
- **📱 Fully Responsive**: Optimized for all devices—mobiles, tablets, and desktops.
- **🎨 Modern UI**: Sleek dark-mode aesthetic with vibrant orange accents.

---

## 🚀 Technologies Used

- **HTML5**: Semantic and accessible document structure.
- **CSS3**: Custom properties (CSS variables), Flexbox, and responsive layout techniques.
- **JavaScript (Vanilla)**: For form logic, automated date restrictions, and WhatsApp redirection.
- **FontAwesome**: For beautiful, modern iconography.
- **Google Fonts**: Clean and professional typography.

---

## 📂 Project Structure

The project starts here:
- **`index.html`**: The main landing page (HTML, CSS, JS).
- **`README.md`**: Project documentation.

---

## 🛠️ How to Use

1. **Clone/Download**: Download the repository or specific files.
2. **Open**: Double-click the `index.html` file to open it in any modern web browser.
3. **Browse & Book**:
   - Scroll through the available bikes.
   - Check the rental rates in the "Offers" section.
   - Fill out the "Book Your Bike" form and click "Book via WhatsApp".

---

## 🎨 Customization

To customize the website for your own needs:
- **Change Colors**: Locate the `:root` variables in the CSS section of `index.html`.
- **Update Phone Number**: Change the `phone` constant in the JavaScript section of `index.html` to your own WhatsApp number.
